# Specification

## Summary
**Goal:** Build a modern, professional, responsive multi-page marketing website for “Nath Cyber Cafe” with Home/About/Services/Contact pages, a clear services presentation, WhatsApp/contact options, and SEO-friendly metadata.

**Planned changes:**
- Create responsive site layout using a clean minimalist design with royal blue/cyan/white palette and modern sans-serif typography.
- Implement routing for Home, About Us, Services, and Contact with a consistent header/navigation and footer across pages.
- Build the Home hero section with the exact provided headline/subheading, CTA buttons linking to Contact and Services, and a computer/digital-themed illustration asset.
- Add a Home services preview and a detailed Services page showing all provided categories and service items with the same wording.
- Add a “Why Choose Us” section with the four provided points.
- Implement the Contact page/section showing the business name and phone numbers, a WhatsApp chat link, a contact form (Name/Phone/Message) with validation and success/error states, and a Google Map embed placeholder.
- Add footer (and/or contact area) social icons/links with centralized configurable placeholder URLs.
- Add SEO structure: semantic headings, per-page titles/meta descriptions, Open Graph/Twitter tags, and LocalBusiness JSON-LD for “Nath Cyber Cafe” with the provided phone numbers.
- Add a Motoko backend endpoint to accept and persist contact form submissions in canister state.

**User-visible outcome:** Visitors can browse a polished 4-page website for Nath Cyber Cafe, view categorized services, see “Why Choose Us,” open WhatsApp chat, submit a contact form that is saved on the backend, view a map placeholder, and share pages with proper SEO/social previews.
